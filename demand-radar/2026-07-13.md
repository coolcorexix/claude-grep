### 2. [branching] GitHub #55347 — anthropics/claude-code — May 1, 2026

**"/rewind says 'conversation will be forked' but never creates a resumable fork"**
[https://github.com/anthropics/claude-code/issues/55347]

- **Context:** Claude Code CLI (macOS, confirmed cross-platform). `/rewind` promises to fork the session but instead mutates the same JSONL in-place — no new session ID, no new file. The original end-state disappears from `/resume` and is permanently unreachable.
- **Pain tag:** [branching] — native fork is broken, data loss on attempted branch
- **Opportunity:** high — the bug is open and assigned; Ctrl-B in ccfind implements the actual fork-from-message that `/rewind` promises but can't deliver, writing a new independent JSONL. Credibility line: ccfind is `/btw`-compatible (Claude Code's native fork format).

**Drafted reply:**
> "Same in-place mutation behavior here — /rewind touching the existing JSONL means you lose the pre-rewind branch. While this is open: Ctrl-B in `ccfind` (claude-grep) does what /rewind intends — forks the session at any past user message into a new independently resumable JSONL, `/btw`-compatible. Disclosure: my project."
